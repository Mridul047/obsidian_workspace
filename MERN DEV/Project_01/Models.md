## Common DB Models to be implemented:

### user.js
	Contains schema related to user interacting with the system.
	* firstName
	* LastName
	* email
	* password
	* phoneNumber
	* accountType
	* additionalDetails [[#profile.js]]
	* courses [[#courses.js]]
	* courseProgress

### profile.js
	Contains schema related to user profile interacting with the system.
	* image
	* billingDetails
	* dateOfBirth
	* gender
	* profession
	* about

### courses.js
	Contains schema related to user profile interacting with the system.
	* courseName
	* courseDescription
	* instructor
	* whatYouWillLearn
	* courseContent
	* rating
	* price
	* thumbnail
	* tag
	* studentsEnroller
